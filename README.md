# Ecommerce Purchases Project

## Overview

This project analyzes E-commerce purchase data using Python and Pandas in Jupyter Notebook. The main goal of this project is to perform data analysis and extract useful customer insights from the dataset.

## Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## Features

* Data Cleaning and Exploration
* Finding Average Purchase Price
* Analyzing Email Providers
* Filtering Customers by Conditions
* Purchase Pattern Analysis
* Working with String Operations and Functions

## Sample Questions Solved

* What is the average purchase price?
* How many people use Mastercard and purchased above 50?
* Counting people with specific job titles

## Learning Outcome

Improved skills in:

* Data Analysis
* Pandas Operations
* Filtering Data
* Working with Real-world Datasets
* Jupyter Notebook
